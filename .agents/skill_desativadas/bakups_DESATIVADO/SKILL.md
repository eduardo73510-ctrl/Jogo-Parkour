# bakups

Use esta skill sempre que o projeto Construct 3 for atualizado e a alteração envolver o arquivo `.c3p`.

## Pasta principal do projeto

O projeto fica em:

`C:\DOCUMENTOS\Jogo`

Antes de qualquer comando, entrar na pasta:

```bat
cd /d C:\DOCUMENTOS\Jogo
